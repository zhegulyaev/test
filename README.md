# Тарифные планы

Простой прототип страницы настройки тарифов.

## Просмотр

```bash
python -m http.server 8000
```

После запуска откройте `http://localhost:8000` и выберите нужный тариф. Галочки минималистичные, подсказки поясняют выгоду и аудиторию тарифов, а количества элементов можно менять прямо в карточках.
